1. Mesh2inp.py => for converting .mesh with 2D quad elements (e.g. hex 2by 2 mesh)
2. Mesh2inp_3d.py => for converting .mesh without 2D elements (e.g. bunny.mesh)
3. Inputs: 
		new1.mesh=> hex 2 by 2 by 2 mesh downloaded from the website (for Mesh2inp.py)
		bunny.mesh=> bunny mesh downloaded from the website (for Mesh2inp_3d.py)
5. outputs: 	new1.inp from Mesh2inp.py
		bunny.inp from Mesh2inp_3d.py
